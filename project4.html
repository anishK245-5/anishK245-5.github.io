Project 4: Environmental Conciousness Quiz

About the Project

In this project me and my partner CJ Pabalan created a quiz that counts your score based on the answers you select. It asks questions based on what you would do for the environment and you have to try to make the right choice. The lower the score, the more you are likely to help the environment.

Concepts Learned

  • Using mouse and keyboard interatctions
  • Using if-elif-else statements in python
  • Testing the user's contribution towards a better world
