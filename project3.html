Project 3: Tree Planting Game

About the Project

In this project me and my partner CJ Pabalan created a game where you are supposed to raise money and buy seeds to clear the polluted sky. This addresses the age old problem of global warming which we gave the simple solution to raise money and plant seeds to contribute to help the environment.

Concepts Learned

  • Using mouse interatctions
  • Making parameters for the mouse clicks to do different actions
  • Spreading a good messsage in a fun and simple way
