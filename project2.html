Project 2: Global Hunger Awareness

About the Project

I created an interactive slideshow using code to spread awareness for global hunger. It presents the problem of world hunger and explores possible solutions. The goal was to express these challenges visually using symbolic shapes, meaningful text, and transitions.

Concepts Learned

  • Using functions to organize code
  • Making the program interactive using conditions and user input
  • Layering shapes and text to simulate slides and transitions
