Project 2: Global Hunger Awareness

About the Project

I created an interactive slideshow using code to spread the awareness for Global Hunger. It presents the problem of global world hunger and solution. The goal was to express these global challenges through symbolic shapes and clear labels.

Concepts Learned

  • Included more interactiveness
  • Learned how to program code with mouse buttons
