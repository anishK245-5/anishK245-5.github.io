Project 5: Food Delivery Game

About the Project

In this project me and my partner CJ Pabalan created a game that counts your pollution score based on how many times you press the arrow keys to drive the car. Your main goal is to collect food packages scattered on the map and try to bring them back into the building. The sample code written down at the bottom makes the game draw the rocks and bushes.

Concepts Learned

  • Using mouse and keyboard interatctions
  • Using "or" and "and" statements in python
  • Creating a fun way to promote Hunger Awareness

 def drawRock(rockX,rockY):
    rockGroup=Group(
        Polygon(65,60,90,65,95,105,55,105,fill=rgb(90,90,90)),
        Polygon(58,77,74,70,83,105,55,105,fill='slateGrey')
        )
    rockGroup.centerX=rockX
    rockGroup.centerY=rockY

def drawBush(bushX,bushY,berries):
