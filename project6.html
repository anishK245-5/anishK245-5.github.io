Project 6: Affordable and Clean Energy Games

About the Project

In this project me and my partner CJ Pabalan created two games. One built by me and one built by CJ. CJ's game is on clean electricity. The sun is presented, and your cursor has to be on the sun to gain electricity. After a certain amount of energy points, the game will congragulate you. I created a game where there is a fueling station with a sign saying "Renewable Fuel". I am a big believer in renewable sources of fuel and I am able to show it in this game.

Concepts Learned

  • Making animations by using onStep function
  • Keeping the same interactiveness as my other games
  • Creating a fun way to promote Clean and afffordable energies

sustainableFuelCenter = Group(Star(150,375,26,6,roundness=85,fill='lightGrey',border='darkBlue',borderWidth=5,rotateAngle=30),
    visible=False) 
fuelSign=Group(
    Line(330,400,330,250,lineWidth=5),
    Rect(250,175,150,75,fill='lightGrey',border='black'),
    Label('Renewable Fuel!',330,215,bold=True,size=15),
    visible=False
    )
fuelNozzle=Line(176,375,179,380,visible=False)
fuelCounter=Label(0,200,100,bold=True,size=15)
truckCounter=Label('Cars Served:',330,230)
truckCounterNumber=Label(0,375,230)
truck=Group(
    Polygon(230,360,240,360,250,375,250,380,230,380,230,375,fill='brown'),
    Rect(233,365,8,6,fill=gradient(rgb(150,150,150),'grey', rgb(150,150,150),'grey',start='left')),
    Rect(195,371,35,9,fill=gradient(rgb(150,150,150),'grey', rgb(150,150,150),'grey',start='left')),
    Circle(240,385,7),
    Circle(210,385,7),
    Circle(240,385,3,fill='grey'),
    Circle(210,385,3,fill='grey'),
    visible=False
    )
game2Screen=Group(sustainableFuelCenter,truck,fuelSign,fuelNozzle,fuelCounter,truckCounter, truckCounterNumber,visible=False)
def pageMover(page):
      game2Screen.visible=True
def onStep():
      if game2Screen.visible==True:
        if fuelCounter.value>=100:
            truck.left+=1
        if truck.right>=400:
            truck.centerX=210
            fuelCounter.value=0
            truckCounterNumber.value+=1
        if truckCounterNumber.value==10:
            winScreen(2,True)
def onMousePress(mouseX,mouseY):
      if game2Button.contains(mouseX,mouseY):
