Project 1: Illinois Global Scholar

About the Project

I created a basic visual for the Illinois Global Scholar program. It represents ideas for improving the world, including solutions like Sustainable Fuel and Fair Pay. The goal was to express these global challenges through symbolic shapes and clear labels.

Concepts Learned

  • Drawing basic shapes (Rectangles, Circles, Lines)
  • Positioning and customizing text with Labels
  • Layering and aligning visual elements

Sample Code

Rect(15,235,75,65,fill='crimson')
Line(15,235,90,300,fill='red',lineWidth=5)
Line(90,235,15,300,fill='red',lineWidth=5)
Label('Sustainable',130,215, size=13, fill='white', font='arial', bold=True)
Label('Fuel',130,228, size=13, fill='white', font='arial', bold=True)
Rect(30,120,80,40, fill='green')
Circle(70,140,18, fill='mediumSeaGreen')
Label('$100',41,155,size=10)
Label('$100',99,125,size=10,rotateAngle=180)
